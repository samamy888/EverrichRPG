## 使用原則

- 遊戲採 `tile_mode + layered_tilemap`，照片用來推導空間語言，不直接當成地圖背景。
- 地板、牆面與通道做成可重複 16×16 tiles；櫃台、店門、指示牌與展示架維持獨立物件。
- 照片中的人物、臉部、廣告畫面與現場文字不進入遊戲素材。
- 品牌標誌只作店型辨識參考；正式使用前仍需通過品牌授權清單。
- 實景透視不可直接換算碰撞範圍，碰撞與通行寬度以遊戲可讀性重新設計。

## 可形成的遊戲區域

### 1. 中央免稅中庭

參考照片：

- `150159`
- `150202`
- `150215`
- `150222`
- `150613`
- `150620`
- `150723`

建議結構：

- 中央保留 6–8 tiles 寬主走道。
- 左右配置開放式酒類、精品與伴手禮店面。
- 大型電子地圖作為地標與可互動導引設施。
- 手扶梯／中庭邊緣作為不可通行邊界或未來區域入口。
- 使用灰色磨石主地板、暖白店內地板及深色店面邊框。

### 2. 美妝精品長廊

參考照片：

- `150755`
- `150758`
- `150800`
- `150804`

建議結構：

- 中央使用 8–10 tiles 寬藍灰地毯。
- 左右店面採半開放櫃位，不必每間都切換地圖。
- 重點店鋪才使用黑幕切換成獨立室內場景。
- 圓形天花燈飾可做成固定環境裝飾，不參與碰撞。

### 3. 精品電動步道走廊

參考照片：

- `150924`
- `150954`

建議結構：

- 一側為電動步道，另一側為精品店面。
- MVP 尚未做航班功能，因此只作商店街的連接區。
- 電動步道初期可封閉或作靜態裝飾，避免先增加移動系統複雜度。
- 大型方向牌可作區域出口提示，不顯示真實登機門資料。

### 4. 餐飲與休息支線

參考照片：

- `150306`
- `150308`
- `150933`

建議結構：

- Phase 2 暫不製作完整區域。
- 可先抽取桌椅、餐飲櫃台與候機座椅，作日後 NPC 支線與恢復／休息功能。
