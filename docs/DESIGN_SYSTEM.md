# 設計系統

## 原則

- 先建立資訊層級，再添加裝飾。
- 顏色、字型、間距與動態必須使用集中 token。
- 所有互動需支援鍵盤、觸控與清楚的焦點狀態。
- 遊戲畫面與管理後台可以有不同語氣，但共用可讀性與無障礙標準。

## 初始 token

- 主背景：`#0f172a`
- 表面：`#111c33`
- 主文字：`#f8fafc`
- 次文字：`#94a3b8`
- 品牌色：`#38bdf8`
- 成功色：`#34d399`
- 圓角：8 / 12 / 16 px
- 間距：4 / 8 / 12 / 16 / 24 / 32 / 48 px

這些 token 是架構基線，不代表新版遊戲最終美術方向。
